## Local Development

To run this website locally:

### Using Docker (Recommended)
```bash
docker compose up
```
Then visit `http://localhost:4000`

### Using Ruby/Jekyll
1. Install Ruby and Bundler
2. Run:
   ```bash
   bundle install
   bundle exec jekyll serve
   ```
3. Visit `http://localhost:4000`
